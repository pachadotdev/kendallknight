About

Please read my article for the full details of this project (Open Access):

Vargas Sepulveda, Mauricio. 2025. ‚ÄòKendallknight: An R package for efficient implementation of Kendall‚Äôs correlation coefficient computation‚Äô. PLOS ONE 20 (6): e0326090. https://doi.org/10.1371/journal.pone.0326090.

This package implements a different algorithm from the one implemented in base R, and it reduces the complexity of the Kendall‚Äôs correlation coefficient from O(n^2) to O(n log n) resulting in a runtime of nano seconds or minutes instead of minutes or hours. This package is written in C++ and uses cpp11 to export the functions to R. See the vignette for the mathematical details.

Installation

You can install the released version of kendallknight from CRAN with:

install.packages("kendallknight")

You can install the development version of kendallknight like so:

remotes::install_github("pachadotdev/kendallknight")

Examples

See the documentation and vignette: https://pacha.dev/kendallknight/.
