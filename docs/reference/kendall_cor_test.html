Description

Kendall correlation using the exact values when the number of observations is less than 50. For larger samples, it uses an approximation as in base R. The possible values are "two.sided", "greater", and "less". Must be a single number between 0 and 1. Default is 0.95. \item{statistic}{The Kendall correlation coefficient.} \item{p_value}{The p-value of the test.} \item{alternative}{A character string describing the alternative hypothesis.} Kendall's Tau with Ungrouped Data".
